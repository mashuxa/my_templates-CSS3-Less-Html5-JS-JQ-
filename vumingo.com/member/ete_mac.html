What is ETE Player?

ETE Player is a test engine designed specifically for certification exam preparation. It allows you to take practice tests in an environment very similar to an actual exam.

  • ETE Player organizes the exams created in ETE Designer and presents them to the exam taker. The ETE Player interface functions is a realistic simulation of the actual certification exam.
