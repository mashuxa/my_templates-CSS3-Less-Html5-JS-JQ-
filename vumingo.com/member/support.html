Support

Most questions about our software have already been answered. Please check our knowledge base for answers to your most likely questions.

  • Using ETE Exam Simulator
    • Where can I find exams?
      We do not provide any practice exams directly. Many exams, however, are available on the Internet. For example, try searching for a term such as "ccna vce".
    • I have received an email: 'Please note that you are close to exceeding the limit on the number of allowed activations for your license'. Will my license be suspended?
      You need to contact customer support. Please use the form on that page, or email us, and include your full name and the e-mail address that you used when making your purchase.
    • How can I activate VCE Exam Simulator?
      Use E-mail and Password of your Avanset.com member account. That is the only information required to activate VCE Exam Simulator that you purchased.
    • How many computers can I run VCE Exam Simulator on?
      By default VCE Exam Simulator can run on 2 different computers. However you can purchase additional licenses and run it on up-to 10 computers.
    • Where can I find manual for VCE exam simulator?
      Please consult the extensive help files included with the program. Run Player, then click the Help menu, and then Contents.
    • How do I backup my data?
      We strongly advise you to backup your results, history, and other testing data. By default, all user data can be found in the following locations:

      Windows XP:
      C:\Documents and Settings\All Users\Application Data\VCE Exam Simulator\vcesimulator.db

      Windows 7:
      C:\Users\All Users\VCE Exam Simulator\vcesimulator.db

    • How do I run VCE Exam Simulator on a Mac OS X computer?

      Mac OS X users can now open .vce exam files using VCE Exam Simulator for Mac:
      https://itunes.apple.com/us/app/vce-exam-simulator/id815531296

      Note: VCE Simulator supports most types of questions. Advanced question types (such as Create a Tree, Build List, Drop and Connect) are supported only by VCE Exam Simulator for Windows. To access advanced question types, many customers successfully run VCE Exam Simulator on their Mac within a virtual machine. This requires the use of software such as Crossover, VirtualBox, VMWare Fusion or Parallels.

      The subscriptions at Avanset.com are recurring i.e. after a certain period of time according to the selected package (1 month, 3 months, 6 months or 1 year) it is re-billed automatically by the payment processor.

  • Using VCE Exam Simulator for Mobile
    • How do I add VCE exam files to my iPhone or iPad?
      via iTunes:
      From your computer:
      1. Download the latest update of VCE Exam Simulator for Mobile for iPhone / iPad
      2. Download the latest update for iTunes
      3. Update your iPhone software to 6.0 or later
      4. Choose your iPhone Apps in iTunes and use the drag-and-drop function to sync VCE exam files from your computer to the iPhone/iPad
      Via upload.avanset.com:
      From your computer:
      1. Visit our website upload.avanset.com
      2. Click "Choose" to browse your computer for the VCE exam file to be uploaded
      3. Click "Upload"
      4. Save the file code that is shown to you when the upload finishes
      Then, from VCE Exam Simulator for Mobile for iOS:
      1. Tap the "+" symbol in the upper right corner
      2. Tap "Upload via web"
      3. Enter the file code you saved earlier
      Via Dropbox.com:
      1. Create a free account at Dropbox.com
      2. Upload your VCE exam files to your dropbox account
      3. Download those files to your mobile device using the free Dropbox app available in the Apple App Store
    • How do I add VCE exam files to my Android phone or tablet?
      Via direct file transfer:
      From your computer:
      1. Some Android devices allow direct file transfer from the user's computer. Consult your device's manual for instructions.
      Then, from VCE Exam Simulator for Mobile for Android:
      1. Tap the "+" symbol in the upper right corner
      2. Tap "Local file" and browse to location of the VCE file on your device's memory card
      Via upload.avanset.com:
      From your computer:
      1. Visit our website upload.avanset.com
      2. Click "Choose" to browse your computer for the VCE exam file to be uploaded
      3. Click "Upload"

      Then, from VCE Exam Simulator for Mobile for Android:

      Via Dropbox.com:
      1. Create a free account at Dropbox.com
      2. Upload your VCE exam files to your dropbox account
      3. Download those files to your mobile device using the free Dropbox app available through Google Play
  • Converting VCE Files
    • How do I convert a VCE file to PDF?
      You need to open your VCE file in VCE Designer and print it to a virtual PDF printer. Please type the following into Google for more information: printing to PDF. An example of software that allows for 'print to PDF' functionality can be found at www.nitroreader.com.
